## PART 2 — Exporting Power BI Dashboard Screenshots

> Before pushing to GitHub, export professional screenshots of all 6 dashboard pages.

### How to Export in Power BI Desktop

**Option A — Export to PDF (best quality)**
1. Go to **File → Export → Export to PDF**
2. This exports all pages in one PDF
3. Open the PDF → screenshot each page at full resolution

**Option B — Screenshot each page manually**
1. Press `F11` to enter full-screen mode in Power BI
2. Use `Windows + Shift + S` (Windows Snipping Tool) to capture each page
3. Save as PNG at maximum resolution

**Option C — Export as PowerPoint**
1. Go to **File → Export → Export to PowerPoint**
2. Each slide will be a high-res PNG version of each dashboard page

### Naming Convention for Screenshots
```
page1_executive_overview.png
page2_sales_deep_dive.png
page3_customer_analysis.png
page4_operations_logistics.png
page5_geographic_analysis.png
page6_forecast_2025.png
```
